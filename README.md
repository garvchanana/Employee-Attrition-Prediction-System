# Employee-Attrition-Prediction-System
A Machine Learning–based predictive system that identifies employees at risk of leaving an organization by analyzing HR attributes such as job satisfaction, overtime, income, tenure, work–life balance, and more.
This project also includes a Power BI Dashboard for business insights and visual analytics.
# Project Overview
Employee attrition is a major challenge in HR management. High turnover impacts productivity, increases recruitment costs, and affects long-term organizational growth.
This project uses Machine Learning & Ensemble Stacking to predict attrition and help HR teams take proactive retention measures.

The project includes:

Data Preprocessing

Exploratory Data Analysis (EDA)

Feature Engineering 

Model Training (LR, DT, RF, SVM, GB) 

Ensemble Stacking 

Evaluation Metrics 

Power BI Dashboard

Final Report.
# Dataset Description
Dataset: IBM HR Analytics Employee Attrition Dataset
Records: 1,470 employees
Features: 35 (numerical + categorical)
